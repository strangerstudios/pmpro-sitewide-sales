Run a sitewide sale (Black Friday, Cyber Monday, etc.) with Paid Memberships Pro.

== Description ==

This plugin requires Paid Memberships Pro to function.

The Sitewide Sale Add On allows you to create flash or sitewide sales. A settings page allows you to create a new sale, assign a discount code and unique landing page for the promotion. The plugin will automatically apply a discount code for users who visit the sale landing page. 

The plugin also adds the option to display sitewide page banners to advertise your sale.

== Installation ==

1. Upload the `pmpro-sitewide-sale` directory to the `/wp-content/plugins/` directory of your site.
1. Activate the plugin through the 'Plugins' menu in WordPress.
1. Navigate to the `Sitewide Sale` settings page by going to `Memberships > Sitewide Sale`.
1. Choose a discount code, sale page and banner settings.
